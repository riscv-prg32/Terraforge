# Copyright and clean-room statement

TerraForge 32 is inspired only by broad sandbox mechanics: digging, placing blocks, collecting resources, and surviving a world. These are general gameplay ideas, not copied protected expression.

The repository avoids infringement by using:

- an original title and fictional world premise;
- procedural pixel-art tiles created specifically for this package;
- synthetic WAV beeps generated mathematically;
- original rules, enemies, victory condition, HUD, and map generation;
- no third-party sprites, textures, music, sounds, names, logos, level layouts, story, or UI elements from Minecraft/Mojang/Microsoft or any other commercial game.

Assets are documented in `assets/manifest.json`. Code and assets use separate licenses so the teaching material can be reused safely.
